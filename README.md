# DOFLinx documentation

See [https://doflinx.github.io/docs/](https://doflinx.github.io/docs/).

This is the site of the official DOFLinx documentation  / guide / manual.  DOFLinx is a tool for monitoring pinball and arcade emulators for in game triggers and linking them to output actions.  It works natively with Pinball FX, Pinball M, Infected Mushroom Pinball, Pinball FX3, Pinball FX2, Mame, FBNeo.  For Mame and FBneo it provides in game effects ( IGFX ) also known as Arcade Effects  ( AFX ) and also Retro Game Effects ( RGFX ).
